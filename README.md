# My Static Web App

A simple, clean, and attractive static website built with HTML, CSS, and JavaScript.

## Features

- Responsive layout with header, multiple sections, and footer
- Navigation menu with smooth user experience
- Interactive button with alert
- Contact form with simple validation and confirmation message

## How to use

1. Clone the repository:

   ```bash
   git clone https://github.com/<your-github-username>/my-static-webapp.git
   ```

2. Open `index.html` in any modern browser.

## Project Structure

```
my-static-webapp/
│
├── index.html
├── styles.css
├── script.js
└── assets/
    └── logo.png  (optional)
```
